# hostfunc

> The open-source platform for tiny composable TypeScript functions.

Write a function. Deploy in seconds. Trigger via HTTP, cron, email, or MCP.
Self-host on your own cloud or use the managed version.

**Status:** pre-alpha. Not ready for production.

## Quick start

```bash
pnpm install
pnpm build
pnpm test
```
